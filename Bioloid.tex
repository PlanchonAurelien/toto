\chapter{Introduction}
	\section{Objectifs du projet}
	
	Ce projet consiste en la prise en main du kit Bioloid Premium afin de réaliser un robot hexapode (dont le modèle est proposé dans le kit) et de le programmer pour réaliser des animations en vue d'une démonstration notamment lors des portes ouvertes.
	Ce projet à comme point clés : le montage du robot, la conception et réalisations d'animations, la rédaction détaillée du rapport du projet dans lequel qui pourra resservir de base sur la prise en main du kit pour les projets à venir.

Le kit propose également la construction de 3 bipèdes humanoïdes. Ces robots ont tous la même structure de base. Seul les articulations de la hanche (le liens entre les jambes et le tronc) changent. Le reste de la jambe reste identique peut importe le modèle. De même pour le tronc et les bras. Le manuel de montage des trois kits humanoïdes est disponible dans la documentation papier fournie avec l'achat du kit.

\section{Mise en place du projet}
Pour le projet, nous avons décidé de réaliser le robot King Spider \textbf{LABEL BITCH}. C'est un hexapode comme demandé et il offre l'avantage d'avoir les déplacements ayant le plus d'amplitude parmi les robots proposés.
Nous commencons le projet à partir d'un robot de type humanoide monté l'année précédente par des élèves du parcours PeiP : \href{mailto:mentor.bajraktari@etu.univ-tours.fr}{Bajraktari Mentor} et \href{anthony.demuylder@etu.univ-tours.fr}{Demuylder Anthony}. Nous devons commencer par démonter le robot humanoïde pour monter le King Spider.
